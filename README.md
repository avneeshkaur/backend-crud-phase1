\# Backend CRUD – Phase 1



A Node.js + Express + MongoDB backend project implementing authentication and employee CRUD operations.



\## Tech Stack

\- Node.js

\- Express.js

\- MongoDB (Mongoose)

\- JWT Authentication



\## Features

\- User signup \& login with JWT

\- Protected routes using middleware

\- Employee CRUD operations

\- MVC folder structure

\- Environment variables support
